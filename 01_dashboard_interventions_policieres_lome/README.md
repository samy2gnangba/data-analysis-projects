# Dashboard — Monitoring des Interventions Policières, Grand Lomé

## Contexte
Projet réalisé dans le cadre du **Challenge Data Mensuel Juillet 2025** de Data Community Togo, portant sur l'analyse des interventions policières dans le Grand Lomé.

## Objectif
Construire un tableau de bord Power BI permettant de suivre :
- Le volume d'interventions par commune
- Les temps de réponse
- La répartition et les tendances des incidents dans le temps

## Outils
- Power BI Desktop
- Power Query (nettoyage et transformation des données)

## Contenu
- `dashboard_interventions_lome.pbix` — le fichier Power BI complet

## Compétences démontrées
- Modélisation de données dans Power BI
- Création de KPI et de visualisations interactives
- Analyse géographique et temporelle de données d'incidents
