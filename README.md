# Data Analysis Projects

## À propos

Ce dépôt rassemble mes projets pratiques d'analyse de données, réalisés dans le cadre de ma formation **D-CLIC / OIF Data Analytics** (Excel, Power BI, SQL, Python) et de mes travaux personnels.

Contrairement à [python-learning](https://github.com/samy2gnangba/python-learning) qui documente mon parcours de formation Python, ce dépôt présente mes **livrables et projets concrets** en analyse de données.

## Projets

### 🏆 [Dashboard — Interventions Policières, Grand Lomé](./01_dashboard_interventions_policieres_lome)
Tableau de bord Power BI complet réalisé pour le Challenge Data Mensuel Juillet 2025 de Data Community Togo. Suivi des interventions policières par commune, temps de réponse, tendances temporelles.

**Outils : Power BI, Power Query**

### 📊 [Nettoyage de données — Évaluation Data Analyst](./02_nettoyage_donnees_power_query)
Nettoyage et transformation d'un jeu de données de transactions commerciales : gestion des valeurs manquantes, standardisation, création de colonnes calculées avec des formules Power Query personnalisées.

**Outils : Excel, Power Query (langage M)**

### 🗄️ [Projet SQL — Base de données de gestion d'entreprise](./03_projet_sql_gestion_entreprise)
Conception de schéma relationnel et requêtes analytiques (jointures, sous-requêtes, agrégations) sur une base de données RH/commerciale/projet.

**Outils : SQL**

### 🐍 [Exploration de données — Python](./04_exploration_donnees_python)
Analyse exploratoire d'un dataset de type recensement avec pandas : statistiques descriptives, distribution de variables, filtrage.

**Outils : Python, pandas**

### 📈 [Exercices Excel](./05_exercices_excel)
Travaux pratiques d'analyse de données bancaires.

**Outils : Excel**
