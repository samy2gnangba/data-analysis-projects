# Exploration de données avec Python (pandas)

## Contexte
Premier projet d'exploration de données avec Python, sur un jeu de données de type recensement (âge, profession, éducation, revenu).

## Démarche
1. **Connexion au dataset** — chargement du CSV avec pandas, définition des noms de colonnes
2. **Informations générales** — dimensions du dataset, types de colonnes, valeurs manquantes
3. **Statistiques descriptives globales** — `df.describe()`
4. **Exploration d'une variable** — distribution du niveau d'éducation
5. **Filtrage** — sélection des personnes de plus de 40 ans gagnant plus de 50K

## Outils
- Python
- pandas

## Contenu
- `exploration_donnees_adults.ipynb` — notebook Jupyter/Colab complet

## Compétences démontrées
- Manipulation de DataFrames avec pandas
- Analyse exploratoire de données (EDA)
- Filtrage conditionnel multi-critères
